## Практика по теме "Работа с файлами, i/o операции"\*

\* - задачи сгенерированы AI

**Задача 1: Потоковый фильтр логов с несколькими условиями**
Есть лог-файл, где каждая строка содержит уровень (`INFO`, `ERROR`, `WARN`) и текст сообщения.
Напиши утилиту, которая:

- принимает путь к файлу
- принимает флаг `-level` (можно несколько через запятую)
- принимает флаг `-grep`
- выводит только строки, которые одновременно:
  - соответствуют одному из уровней
  - содержат подстроку  
     Важно: файл может быть очень большим → нельзя читать его целиком.

**Задача 2: Объединение файлов с дедупликацией**
В директории лежат `.txt` файлы со строками.  
Напиши программу, которая:

- объединяет их в один файл
- убирает дубликаты строк
- сохраняет порядок первого появления строки
  👉 Нужно самому продумать структуру хранения.

**Задача 3: Универсальный трансформер потоков**
Реализуй функцию:

```
func Transform(r io.Reader, w io.Writer, fn func([]byte) []byte) error
```

Она должна:

- читать поток
- применять функцию к каждому чанку
- писать результат
  Протестируй:
- файл → файл
- stdin → stdout

**Задача 4: Копирование с прогрессом**
Напиши утилиту копирования файла, которая:

- копирует файл через `io.Copy`
- выводит прогресс в процентах (каждые ~100мс)
- корректно работает с большими файлами
  👉 Нужно подумать, как считать прогресс.

**Задача 5: Поиск самого большого файла**
Напиши программу, которая:

- проходит по директории (без вложенности)
- находит самый большой файл
- выводит его имя и размер
  👉 Используй `os.ReadDir` + `FileInfo`.
