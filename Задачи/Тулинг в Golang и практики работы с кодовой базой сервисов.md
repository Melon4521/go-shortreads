## Практика по теме "Тулинг в Golang и практики работы с кодовой базой сервисов"\*

\* - задачи сгенерированы AI

**Задача 1: CLI-утилита с флагами и подкомандами**  
Сделай CLI-приложение:

```
app stats file.txt
app head file.txt -n 10
```

Реализуй:

- подкоманду `stats` → считает строки/слова/байты
- подкоманду `head` → выводит первые N строк

**Задача 2: Конфигурация через флаги + env**  
Сделай программу, которая:

- принимает `--port`, `--host`
- если флаги не заданы → берет из env
- если нет и там → ставит дефолт
  👉 Нужно продумать приоритет.

**Задача 3: Логгер с уровнями**  
Реализуй простой логгер:

- уровни: DEBUG, INFO, ERROR
- флаг `--log-level`
- сообщения ниже уровня не выводятся

**Задача 4: Профилирование программы**  
Возьми любую из предыдущих задач и:

- добавь `pprof`
- замерь CPU и память
- выведи, где больше всего времени
  👉 Нужно реально разобраться, а не просто подключить.

**Задача 5: go generate**  
Создай файл с директивой:

```
//go:generate ...
```

Сделай генерацию:

- файла с константами
- или простого кода
  👉 Цель — понять workflow.
