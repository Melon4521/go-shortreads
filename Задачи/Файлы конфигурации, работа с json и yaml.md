## Практика по теме "Файлы конфигурации, работа с json и yaml"\*

\* - задачи сгенерированы AI

**Задача 1: Универсальный конфиг с fallback**  
Есть 2 файла:

- `config.yaml`
- `config.json`
  Напиши загрузчик, который:
- сначала пытается YAML
- если нет → JSON
- если нет → ошибка

**Задача 2: Валидация структуры + лишние поля**  
При парсинге JSON:

- запрещай неизвестные поля
- валидируй обязательные поля (например, `port > 0`)

**Задача 3: Частичный JSON-парсер (стриминг)**  
Есть большой JSON:

```
[{user1}, {user2}, ...]
```

Напиши код, который:

- читает его через `json.Decoder`
- не загружает целиком
- фильтрует пользователей по условию

**Задача 4: Кастомный MarshalJSON**  
Создай структуру `User`:

- `Password` не должен попадать в JSON
- `CreatedAt` должен сериализоваться в своем формате
  👉 Реализуй вручную `MarshalJSON`.

**Задача 5: Генерация конфигурации**  
Напиши программу, которая:

- генерирует дефолтный конфиг (JSON или YAML)
- сохраняет его в файл
- затем умеет его читать обратно
