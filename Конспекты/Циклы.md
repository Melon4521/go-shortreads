В Go все необходимые циклы задаются четырьмя разновидностями `for`:
- Базовый for
- for как цикл while
- бесконечный цикл for
- for-range

## Базовый for

```go
sum := 0

// for объявление; условие; действие { ... }
for i := 1; i < 5; i++ {
	sum += i
}

fmt.Println(sum) // 10
```

Здесь объявленная переменная видна только внутри тела и условия `for`

## For как цикл while

```go
n := 1

// for условие { ... }
for n < 5 {
	n *= 2
}

fmt.Println(n) // 8
```

## Бесконечный цикл for

```go
sum := 0

for {
	sum++
	fmt.Println(sum) // Будем «бесконечно» писать числа в консоль
}
```

## Цикл for-range

Для итерации по перебираемым типам есть следующий цикл:

```go

/*
for индекс, значение := range структура данных {
	// Тело цикла
}
*/

for i, ch := range "hello" {
	fmt.Println("Индекс: ", i, ", значение: ", ch)
}
```

Если требуется итерация только по индексам, то второе значение пропускается. А если нужны только значения, то используем: `_, val := ...`

## break/continue, метки

- `break` - прерывает текущий цикл
- `continue` - прерывает текущую итерацию цикла

Метка цикла позволяет прервать его полностью (break) или пропустить итерацию (continue) из вложенного:

```go
sum := 0

outer:
for {
	sum++
	for j := 0; j < sum; j++ {
		if j*sum == 111 {
			break outer // прерываем верхний цикл
		}
	}
}
```
