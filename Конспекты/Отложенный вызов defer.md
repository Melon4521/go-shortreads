## defer

В Go можно отложить выполнение функции под конец работы родительской функции с помощью ключевого слова `defer`. При этом можно работать как уже с объявленными функциями, так и с анонимными.

>[!important]
>Внутри функции можно использовать много вызовов `defer`. Порядок их выполнения будет соответствовать LIFO: выполнение начнется с конца, последний `defer` выполнится первым и т.д

```go
func Example1() {
    for i := 1; i <= 3; i++ {
        defer fmt.Print(i, " ") // используем существующую функцию
        // порядок вывода: 3 2 1
    }
}

func Example2() {
    for i := 1; i <= 3; i++ {
        defer func(n int) { // анонимная функция
            fmt.Print(n, " ")
            // порядок вывода: 3 2 1
        }(i)
    }
}
```

>[!important]
>`defer` срабатывает даже после возникновения фатальных ошибок (паник)
