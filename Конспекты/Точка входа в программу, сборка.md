## Точка входа

Точкой входа любого проекта в Go является файл с указанием пакета main (`main.go` - принятое название точки входа, но оно необязательное)

Базовая структура `main.go`:

```go
package main

// импорты пакетов
import (
	"fmt"
	// ...
)

func main() {
	// ...
}
```

- `package main` - устанавливаем название пакета main - точки входа в программу
- `func main() { ... }` - основная функция, которая вызывается автоматически при запуске 

## Сборка и запуск

- `go build main.go` - сборка исполняемого файла программы под ОС
- `go run main.go` - быстрая сборка + запуск (юзаем на этапе разработки)
- `go fmt ./...` - форматирование файла

Go можно собирать под разные ОС. Для этого нужно указать переменные окружения перед вызовом `go build`:
- `GOOS` - целевая ОС (windows)
- `GOARCH` - архитектура ОС (amd64)

```go
GOOS=windows GOARCH=amd64 go build main.go
```
