## Примитивные типы данных

- `bool` - true/false
- `string` - строка, только двойные кавычки
- `int`, `int8`, `int16`, `int32`, `int64` - целые числа, чем больше разрядность типа (число в битах), тем большее число можно хранить и тем больше памяти выделяется
- `uint`, `uint8`, `uint16`, `uint32`, `uint64` - незнаковые целые числа (положительные)
- `byte (uint8)` - байт
- `rune (int32)` - символ в кодировке utf8
- `float32`, `float64` - числа с плав. точкой; `float32` (6–7 значащих цифр) и `float64` (15–17 значащих цифр)
- `complex64`, `complex128` - комплексные числа

> [!info]
> У всех типов по умолчанию есть значения по умолчанию (zero-value):
> `string` "", `int` 0, `bool` false и т.д.

## Объявление переменных/констант

### Переменные

Вариант 1. Указываем **ключевое слово**, **название** и **тип данных**:

```go
var name string
var surname string = "Ivanov"
```

Если сразу инициализируем со значением, то указание типа можно пропустить:

```go
var surname = "Ivanov" // string
```

Присвоение значения:

```go
name = "Ivan"
```

> [!info]
> Если переменной при объявлении не задать значение, то ей присвоится zero-value, которое зависит от её типа

Вариант 2. Множественное объявление

```go
var i, j, k int = 1, 2, 3
var (
	name string = "Ivan" // запятая не нужна
	surname string = "Ivanov"
)
```

Вариант 3. Указываем **название := значение**, тип данных определится по значению

```go
surname := "Ivanov"
```

### Константы

**Константы** - неизменяемые переменные, их объявление совпадает с вариантам 1 и 2 для переменных, только вместо var пишем **const**

```go
const Pi = 3.14159
const name string = "My name"
const surname = "My surname"
const (
	StatusOK       = 200
	StatusNotFound = 404
)
```

### Области видимости

**Область видимости** - область, где переменная доступна.
У функций, условий, циклов есть блоки кода (тело) `{ ... }`, внутри него своя локальная область. Переменные, объявленные внутри блока кода, вне этого блока недоступны.

```go
func hello() {
	var name = "Ivan"
	
	fmt.Println(name) // Ivan
}

fmt.Println(name) // ошибка, name отсюда недоступна
```

Основной поток кода - глобальная область, переменные отсюда видны во всех вложенных блоках кода (в функциях, условиях) при условии, что внутри них не объявлены переменные с таким же именем (затенение):

```go
var name = "Peter"

func hello() {
	var name = "Ivan"
	
	fmt.Println(name) // Ivan
}
```


> [!warning]
> Вне блоков кода (в основном потоке, т.е в глобальной области) способ объявления переменной через **:=** не cработает:
> 
> ```go
> package main
>  
> name := "Ivan" // ошибка
> 
> func main() {}
> ```

## Приведение типов

Для приведения одного типа к другому используются функции, названия которых совпадают с названием нужного типа:

```go
var i int = 15
var f float32 = float32(i)
```
