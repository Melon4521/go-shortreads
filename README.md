Репозиторий содержит мои личные конспекты по темам языка **Go**, составленные на основе курса **"Основы разработки на Go"** в Центральном университете.

## 🎯 Зачем этот репозиторий?

- **Для себя:** Я веду конспекты, чтобы лучше структурировать знания и глубже разобраться в синтаксисе и особенностях Go.
- **Для сообщества:** Конспекты оформлены в формате **шортридов** - кратких выжимках по основным лонгридам курса. Они могут быть полезны другим студентам для быстрого повторения материала и подготовки к контрольным.
