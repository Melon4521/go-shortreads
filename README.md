# Конспекты по Go: Основы разработки на Go (ЦУ)

Репозиторий содержит мои личные конспекты по темам языка **Go**, составленные на основе курса **"Основы разработки на Go"** в Центральном университете.

## 🎯 Зачем этот репозиторий?

- **Для себя:** Я веду конспекты, чтобы достичь отличного понимания Go и его особенностей, для заполнения пробелов в знаниях, которые часто появлялись у меня при прочтении лонгридов (в них есть недостатки).
- **Для сообщества:** Конспекты оформлены в формате **шортридов** - выжимок из лонгридов, в которых я постарался объяснить все моменты понятным (в первую очередь для себя, но надеюсь и вам будет понятно) языком. Основная польза - более глубокое понимание материала, которое могло не сформироваться при прочтении лонгридов.
